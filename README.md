## ⚙️ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 📝 Edit tasks inline
- 🗑️ Delete individual tasks
- 🔄 Reset all tasks with one click
- ❌ Error handling for empty inputs
- 📱 Responsive and minimal UI
- 🎨 Background image and monospace theme
